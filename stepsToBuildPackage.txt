4. Fix turbo.json — ensure types is built first
json{
  "tasks": {
    "build": {
      "dependsOn": ["^build"],
      "outputs": ["dist/**"]
    }
  }
}
The ^build means "build my dependencies first" — so @repo/types will always be compiled before your backend.
